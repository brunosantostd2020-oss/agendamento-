Recursos
Tudo que você precisa.
Nada que atrapalha.
01
Página pública por negócio
Cada cliente tem sua própria página com cores, identidade e link exclusivo para compartilhar.
02
Lembrete via WhatsApp
Envie a mensagem de confirmação com um clique direto no WhatsApp do cliente, com os dados completos.
03
E-mail automático
Confirmação enviada automaticamente ao cliente com link para cancelar se precisar.
04
Avaliação pós-atendimento
Após o atendimento, o cliente recebe um link para avaliar. Você vê as notas e comentários no painel.
05
Lista de espera
Horário lotado? O cliente entra na lista e recebe aviso automático quando uma vaga abre.
06
Relatório mensal
Todo mês você recebe por e-mail: total de atendimentos, faturamento, serviços mais pedidos e avaliação média.
07
QR Code exclusivo
Gere o QR Code e cole no balcão, na vitrine ou nos materiais do negócio para facilitar os agendamentos.
08
Cancelamento pelo cliente
O cliente cancela pelo link sem precisar ligar. A vaga abre automaticamente para novos agendamentos.
09
Histórico de atendimentos
O cliente consulta todos os agendamentos anteriores pelo e-mail, sem precisar de login.
